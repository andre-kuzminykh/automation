Субъект, который воспринимает среду, рассуждает, принимает решения и действует для достижения цели.

Свойства ИИ-агентов

Автономность

Работает без постоянного контроля человека.

Реактивность

Реагирует на изменения в среде и триггеры.

Проактивность

Инициирует действия для достижения цели.

Обучаемость

Улучшается на основе опыта и фидбека (Memory).

Взаимодействие

Общается с людьми, системами и другими агентами.

Мультимодальность

Понимает текст, голос, изображения, таблицы.

Пример: Организация командировки

Ассистент: Вы просите найти билеты, он выдает список.
Агент: Знает расписание, бюджет и предпочтения. Сам мониторит цены, бронирует оптимальный вариант, добавляет событие в календарь и пишет письмо с деталями.

Модель создания ценности

Фокус курса

RUN

Стабильные процессы

Задача: Эффективность, снижение костов, скорость.

Неопределенность: Низкая.

Роль ИИ: Автоматизация рутины, агенты-исполнители, контроль качества.

CHANGE

Трансформация

Задача: Создание новых продуктов, улучшение клиентского опыта.

Неопределенность: Средняя.

Роль ИИ: R&D ассистенты, анализ трендов, персонализация.

DISRUPT

Инновации

Задача: Новые бизнес-модели, смена парадигмы рынка.

Неопределенность: Высокая.

Роль ИИ: AI-Native продукты, полностью автономные сервисы.

В этом курсе мы фокусируемся на измеримом эффекте в RUN-процессах.

Типы AI-компаний

AI-Driven

Роль ИИ: Инструмент (как Excel)
Роль человека: Делает работу, использует ИИ для ускорения.
Интеграция: Точечная (Copilots).
Пример: Дизайнер использует Midjourney.
Цель трансформации

AI-First

Роль ИИ: Операционная модель.
Роль человека: Архитектор процесса, контролер ИИ-агентов.
Интеграция: Системная (Агенты в процессах).
Пример: Клиентская поддержка полностью работает на ИИ, люди разбирают только сложные эскалации.

AI-Native

Роль ИИ: Ядро бизнеса.
Роль человека: Создатель алгоритмов.
Интеграция: Изначальная, бизнес невозможен без ИИ.
Пример: OpenAI, Perplexity, ИИ-стартапы.
